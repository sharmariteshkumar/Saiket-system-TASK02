# 📝 React To-Do App

A modern and responsive **React To-Do Application** developed using **React.js**, **JavaScript**, **CSS**, and **Vite** as part of my **Full Stack Development Internship at Saiket Systems**.

---

## 🚀 Features

* ➕ Add New Tasks
* ✏️ Edit Existing Tasks
* 🗑️ Delete Tasks
* ✅ Mark Tasks as Completed
* 📊 Total, Completed & Pending Task Counter
* 💾 Local Storage Support (Tasks remain after page refresh)
* 📱 Fully Responsive Design
* ⚡ Fast Performance with Vite

---

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6+)
* CSS3
* Vite

## ▶️ Getting Started

### Open the Project

```bash
cd task3-react-todo
```

### Install Dependencies

```bash
npm install
```

### Run the Project

```bash
npm run dev
```

Then open:

```text
http://localhost:5173
```
